# [Spacing](https://navi.whatfix.com/390bbdcd7/p/66d343-spacing)
Spacing is a unit of whitespace used to separate visual elements from each other and organize them.  

Spacing Constants
Navi's components follow 16px spacing system. 

We have defined a set number of spacing constants built off of this memorable number that allow us to organize content in a consistent and efficient way. This helps us build cohesion across our brand. 

As we build out the system, we will apply and standardize these constants to define specific grids, margins, and paddings.
